Detailed Description

template<typename Number, typename Converter, class Setting>
class hypro::reachability::hypro::BoxT< Number, Converter, Setting >

The class which represents a box.

A box is represented by two points, its maximal and its minimal vertex.

Template Parameters
NumberThe used number type.
ConverterThe used converter. @{

Converts the current box from its two-point representation into a vertex representation, i.e. a V-polytope. The representation is generated by creating all possible combinations of upper and lower bounds of the box.

Returns
A vector of points.

Implements hypro::reachability::hypro::GeometricObject< Number, BoxT< Number, Converter, Setting > >.
