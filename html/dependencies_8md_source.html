1 Dependencies {#dependencies}
2 ============
3 
4 The very basic dependencies include the libraries.
5 
6 - a C++ compiler, g++ (version >= 4.9) or clang++ (version >= 3.6),
7 - [gmpxx](https://gmplib.org/) the Gnu Multiprecision library C++ interface,
8 - [CMake](https://cmake.org/) (version >= 3.0.0), a build system,
9 - [boost](http://www.boost.org/) (version >= 1.55), a collection of C++ libaries,
10 - [CArL](https://github.com/smtrat/carl), a C++ library for arithmetic,
11 - Java JRE (required for the Antlr parser) and
12 - uuid-dev (required for the Antlr parser)
13 
14 Note that CArL installs dependencies such as Eigen3, gmp and boost locally in its build folder, so installing Java and uuid-dev along with CArL should be sufficient to be able to build and use HyPro.
15 To compile HyPro, a suitable C++ compiler is required, either g++ (at least version 4.9) or clang++ (at least version 3.5).
16 
17 Additional (optional) dependencies
18 ----------------------------------
19 
20 Currently HyPro provides an extended interface for linear solving, which allows to _additionally_ add __one__ of the following linear optimizers:
21 
22 - [SMT-RAT](https://github.com/smtrat/smtrat)
23 - [SoPlex](http://soplex.zib.de/)
24 - [Z3](https://github.com/Z3Prover/z3)
25 
26 Using one of these solvers can be enabled by the respective cmake flag , i.e. HYPRO_USE_SMTRAT to enable SMT-RAT support. When using one of the previously mentioned solvers, the usage of rational types such as mpq_class is improved, as linear optimization now can be performed providing _exact_ results.
