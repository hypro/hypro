bool hypro::Dictionary< Number >::fixOutOfBounds ( )

Check for the satisfiability of the dictionary and finds a suitable assignment.

Checks if variables are out of bound. If yes, tries to find a suitable pivot to fix this. If there is none throws exception (empty set). If a suitable pivot is found, apply pivot step and update assignment.

Returns
True, if assignment has been updated.
