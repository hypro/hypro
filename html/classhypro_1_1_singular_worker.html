Detailed Description

template<typename Representation>
class hypro::SingularWorker< Representation >

Class implementing a worker for rectangular automata.

A worker implements a method computing a time transition followed by a discrete transition. The repeated calling of a worker allows to implement a complete reachability analysis method.
