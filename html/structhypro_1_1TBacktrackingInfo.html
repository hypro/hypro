Detailed Description

template<typename Number>
struct hypro::TBacktrackingInfo< Number >

Struct holding all currently required information for backtracking.

Its contents currently are the path (starting and including the current node) and the desired refinementLevel. The Bt-Path also contains the location in which the bad state was hit.
