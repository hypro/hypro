Detailed Description

template<typename Number, typename S = ConstraintSetSettings>
class hypro::ConstraintSetT< Number, S >

A class representing a plain constraint set. This class is intended to serve as a container.

A constraint set in this case is a datastructure similar to a H-Polytope but without any functionality. The idea is just to have a container for data received from the parser or as an intermediate type.
