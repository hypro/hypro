Detailed Description

template<typename Number>
class hypro::HybridAutomaton< Number >

Class for linear hybrid automata.

Template Parameters
NumberNumber type.

The automaton owns all the locations and transitions. Newly created locations and transitions can only be added as unique pointers. If another class/object requests a location/transition, then only normal pointers are returned.

Examples:
example_bouncingBall.cpp.

Function which tests whether this can be the result of a parallel composition with rhs.

Used to verify the result of operator|| for hybrid automata. Checks whether this automaton can be the composed of rhs and some (potentially empty) other composite.
