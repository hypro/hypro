Detailed Description

template<typename Representation>
class hypro::SingularAnalyzer< Representation >

Class which implements a forward reachability analysis method for singular automata.

Requires the usage of a suitable state set representation, e.g., carlPolytope.

Template Parameters
RepresentationThe used state set representation.
