Adds several objects represented as a vector of vectors of points. Duplicates and points inside will be removed by the plotter, which runs a 2d-convex hull algorithm (Graham's scan) one the vertices.

Parameters
[in]_pointsThe points.
Returns
The id of the last added object, which allows to reference the object to change its colors. All other ids can be derived using the size of the passed vector (ids are assigned ascending).
