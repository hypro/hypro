template<typename Number, typename Converter, class Setting>
class hypro::BoxT< Number, Converter, Setting >

A box is represented by an ordered sequence of intervals. Details can be found here.

Evaluation function (convex linear optimization).

Linear optimization for boxes can be done more efficiently that for general convey polytopes as the state space dimensions are independent. Consequently it suffices to consider the coefficient of the cost function for each dimension and either use the upper, respectively lower bound depending on its sign (positive, respectively negative) to get the vertex of the box representing the optimum.
