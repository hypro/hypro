130  // The needed functions for evaluate. Virtual s.t. they can be implemented in the Operation/Leaf classes
131  // Three functions are needed: transform, compute and aggregate.
132  // - transform will be called by every node: Needed parameters are modified via transform
133  // - compute will be called once by every leaf node: Compute computes the result in the leaf noeds from the given parameters and returns it
134  // - aggregate will be called once by every non leaf node: Given the results of its child nodes, a non leaf node A processes these to results to one from A
135  //
136  // Order of calling:
137  // 1) From top to bottom, all nodes call transform
138  // 2) When arriving at the lowest level, leaf nodes first call transform, then call compute
139  // 3) From bottom to top, all non leaf nodes call aggregate
140  //
141  // NOTE: All functions used as the transform must have the signature A name(A param)
142  // All functions used as the compute must have the signature B name(A param)
143  // All functions used as the aggregate must have the signature B name(std::vector<B> param)
144 
145  //For everyone - transform
146  virtual matrix_t<Number> transform( const matrix_t<Number>& param ) const = 0;
147 
148  //For leaves - compute
