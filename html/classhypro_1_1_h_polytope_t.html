Constructor from a matrix and a vector.

Each row in the matrix is considered to represent a constraint, i.e. a normal to a bounding hyperplane. Each corresponding entry in the passed vector is the offset of that plane, i.e. the constant part of the linear constraint.

template<typename Number, typename Converter , class Setting >
hypro::HPolytopeT< Number, Converter, Setting >::HPolytopeT ( const matrix_t< Number > &  A)

Constructor from a matrix.

Each row in the matrix is considered to represent a constraint, i.e. a normal to a bounding hyperplane. The offset of each plane is considered to be 0, i.e. if the number of affinely independent constraints is larger or equal to d+1, the resulting polytope represents the origin point.

Parameters
AA matrix.
